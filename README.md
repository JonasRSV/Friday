## Installation 

Cross compilation have been tested for

* [x] x86 Linux
* [x] Raspberry Pi 3
* [ ] Raspberry Pi 4
* [ ] Windows
* [ ] Mac

Use the build script to compile, see: 

```bash
./build -h
```

For example to build for x86_64 linux

```bash
./build -li
```
